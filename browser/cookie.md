## cookie & session

二者共同写成标记状态。

|        | cookie                 | session    |     |
| ------ | ---------------------- | ---------- | --- |
|        | 服务端种在客户端的标记 | -          |     |
| 存在于 | 客户端                 | 服务端     |     |
|        | 参于通信               | 不参于通信 |     |
| 值     | sessionId              | -          |     |
| 设置   | Set-Cookie             |            |     |
|        |                        |            |     |
