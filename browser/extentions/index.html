    |--xxx

清单 manifest.json

必须在根目录。 开发期间支持注释。上传到市场前必须删除注释。

{
    "manifest_version": 3, // 必须
    "name": "Reading time", // 必须
    "version": "1.0", // 必须
    "description": "str ...",
    "content_scripts": [
        {
            "js": ["script/content.js"],
            "matches": ["<all_urls>"], // 由<scheme>://<host><path>组成
        }
    ],
    "background": {
        "service_worker": "bg.js", // 指定service worker
    },
    "permissions": ["activeTab", "scripting"],
}
