# webgl

## 它是什么

它是基于 open gl 的 3d 绘图协议(浏览器上的 opengl)。底层是 js api。运行在 gpu.

## 可以做什么

使用它在 web 页面上绘制 2d 图片/3d 场景。

## 基本概念

在 gpu 上运行 glsl 语言的代码。它会处理顶点着色器和片元着色器。

## 工作原理

1. 调用顶点着色器，计算出每个顶点的在裁剪空间坐标系中的对应位置。
2. 调用片元着色器计算每个顶点的颜色值。
3. 把颜色值存储在 gpu.
4. 光栅化图形。
