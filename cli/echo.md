# echo
> echo是bash和c shells内置的命令。
> 用于提供标准输出

## feature
- 输出指定字符串
- 输出变量对应的字符串
- 输出转义后的字符串

## usage
```shell
echo [options] [string]

echo string     # string
x=2
echo $x         # 2
echo -e The gif files are *.gif     # 输出当前目录下所有*.gif的文件名
```
