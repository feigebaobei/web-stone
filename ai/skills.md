# skills

> skills（技能）是 anthropic 在 2025.10 推出的能力扩展机制。
> 用于教会 claude 特定的工作方式。（即：传授知识和流程）
> 本质是 md 文档。

## 目录

```
my-skill/
|-- SKILL.md         # 核心指令文件
|-- references       # 参考资料（可选）
    |-- patterns.md
|-- example/         # 示例（可选）
    |-- demo.sh
|-- scripts/         # 辅助脚本（可选）
    |-- validate.sh
```

## 工作原理

1. claude 会扫描所有 skill 的元数据
1. 根据用户请求判断哪些 skill 相关
1. 仅加载相关 skill 的完整内容
