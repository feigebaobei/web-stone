# qwik 的前置环境

水合作用：

- 侦听器 - 定位事件侦听器并将它们安装在 DOM 节点上以使应用程序具有交互性；
- 组件树 - 构造数据并表现在组件树上。
- 应用程序状态 - 恢复应用程序状态。

## 补水过程：

1. 必须下载当前页面相关的所有组件代码。
2. 必须执行与页面上的组件关联的模板，用于重建侦听位置、内部组件树（vdom）
