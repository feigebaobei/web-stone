# `redom`

[redom官网](https://redom.js.org/)

## overview

> 小，只有2k.
> 强。同步处理数据为dom。
> 高性能，使用真实dom，运行快、减小内存。比react快、占内存少。
> 容易学。只需要纯js。

## usage
```shell
npm i redom

import { el, mount } from "redom";

mount(document.body, hello);
```
