# 前端微服务

微服务：几个小的、自主的服务在一起工作。

- 支持不同的团队、不同的技术栈整合为一个应用。
-

## 实现方式

- iframe
- 在 client 使用 js 载入 module
- web components

  - shadow dom
  - html templates
- webpack module federation [ModuleFederationPlugin]()
  - name
  - filename
  - expose
  - shared
