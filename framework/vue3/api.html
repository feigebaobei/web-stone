可分为三类：

  • 响应式 api
  • 生命周期方法
  • 依赖注入，（需要与响应 api 结合使用）

为什么使用组合式 api

  • 更好逻辑复用
  • 方便扩展
  • 更好的类型推断。主要与 ts 结合使用。
  • 打包结果更小
