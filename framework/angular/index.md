# angular

## description

自信地完成 web 应用开发。
创建单页面应用。  
npm 8.5.0+  
基于组件的前端框架。可创建可扩展的前端应用。  
使用 ts 开发。（ng 是 google 出品，ts 是 microsoft 出品）  
内置很多功能。如：router/c&s 通信/形式管理/
内置开发工具。可用于开发、打包、测试。  
基于组件开发  
持续迭代的库

## feature

- 任意扩展
- 打包工具好。还是基于 webpack 搞的。

## setup
