autocannon

overview

node环境的压力测试工具。 支持cli & js

install

npm i autocannon

usage

cli

autocannon [opts] url
url是合法的http(s)的url。
支持环境变量中的端口号。如'http://localhost:$PORT/path'

opts:
-c/--connections N 连接数
-p/--pipelining  N 在每个管道中的语境数量
-d/--duration    秒 运行时间
-a/--amount      N 请求总数。优先级大于duration
-L               N 
-S/--socketPaht
-w/--workers
-W/--warmup
--on-port
-m/--method      METHOD 请求方式。默认'GET'
-t/--timeout     N秒 达到打时长时未连接成功，则重试。默认为10
-b/--body        BODY 请求体。
-F/--form        FORM 用于上传form (multipart/form-data)
-i/--input       FILE 与请求体有关
