@babel/parser

overview

基于acorn/acorn-jsx，在 babel 中做解析器。
把原代码转换为ast

feature

  • 默认支持到 es2020.
  • comment attachment
  • 支持 jsx/flow/ts

install

npm i -D @babel/parser

usage

const @babel/parser = require('@babel/parser');
// or
// import @babel/parser from '@babel/parser';
// TODO: DEMONSTRATE API
