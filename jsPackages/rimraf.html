rimraf

overview

为node环境提供的unix command rm -rf

feature

  • return promise
  • --glob
  • arrays of paths
  • 支持esm/cjs
  • 优先使用原生实现，在win可接收到。原生实现更快、更可靠。
  • 支持cli/api

install

npm i rimraf

usage

// 以esm为例。cjs同理
import rimraf from 'rimraf'
import {rimraf, rimrafSync, native, nativeSync} from 'rimraf'
