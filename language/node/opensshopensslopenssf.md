SSL 是一种国际标准的加密及身份认证通信协议，浏览器就支持此协议。ssl(Secure Sockets Layer)

SSH 的英文全称是 Secure SHell。通过使用 SSH，你可以把所有传输的数据进行加密，这样“中间人”这种攻击方式就不可能实现了，而且也能够防止 DNS 和 IP 欺骗。
