EventSource 也叫 server-sent-event(sse)

  • 由html5提供的api
  • 基于http协议，由server推送给client.
  • client使用事件监听器捕获事件。

client

let eventSource = new EventSource(url)
eventSource.onmessage = (event) => {
    clog(event.data)
}
eventSource.onerror = (event) => {
    clog(`${readyState} = ${event.currentTarget.readyState}`)
}
