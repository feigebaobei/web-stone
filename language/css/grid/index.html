grid 布局

overview

二维布局 google 在 flex 布局出现后，开发了该布局方式。 现在浏览器已经支持此布局方式 适用于大规模布局 由“网格容器”“网格项”共同完成。 网格单元是最小单位。 网格区域都是矩形的。非矩形不成立。

usage

  1. 为容器定义一个网格。
  2. 把网格项指定到占用的网格。 网格决定网格项的排列方式。 网格项决定在单元格内的位置。
