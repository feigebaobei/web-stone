## white-space 处理空白文本的样式

<!-- prettier-ignore-start -->
| |      |        |          |
| --- | -- | ----- | ----- |
| normal 默认值 |      |        |          |
| pre   | 使用等宽字体处理文本 | 不合并空白文本 | 遇到边界时不换行。       |
| nowrap        |      | 合并空白文本   | 不换行   |
| pre-wrap      | 使用等宽字体处理文本 | 不合并空白文本 | 遇到边界时换行。         |
| pre-line      |      | 不保留空白文本 | 保持文本的换行。遇到边界时换行。 |
