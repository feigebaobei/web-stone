typescript

overview

typescript 的包。 在命令行中使用 tsc

feature

  • 根据配置文件转换本地项目。

install

npm i typescript -g

usage

tsc
// 使用配置文件编译
tsc index.ts
// 编译index.ts
tsc src/*.ts
// 编译src目录下的ts文件
tsc --project tsconfig.production.json
// 指定配置文件
tsc index.js --declaration --emitDeclarationOnly
// 指定配置项
// 为js文件设置注解文件
tsc app.ts utils.ts --target esnext --outfile index.js
// 编译为一个js文件。index.js
