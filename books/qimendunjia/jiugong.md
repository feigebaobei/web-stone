# 九宫

通常我们在用奇门遁甲推断事情的发展和未来吉凶的时候，最后会涉及到应期的问题，此时就要通过用神所落的宫位取数来看应期。那么每个宫位对应的数字是怎么来的呢？回答：取自河图洛书、先天八卦和后天八卦，其中洛书对应的数字和后天八卦是一致的，所以我们分为三个部分来讲：河图五行取数、先天八卦取数、后天八卦取数。 1.河图五行取数
（图略）
左边的图案就是河图，我们只需要记一个口诀就可以：

天一生水，地六成之；

地二生火，天七成之；

天三生木，地八成之；

地四生金，天九成之；

天五生土，地十成之。

因此，九宫八卦中坎宫属水对应 1、6，离宫属火对应 2、7，震宫巽宫属木对应 3、8，兑宫乾宫属金对应 4、9，而坤宫艮宫和中宫属土对应 5、10。

2.先天八卦取数

老规矩先来一段口诀：

天地定位

山泽通气

雷风相薄

水火不相射
（图略）
我们可以按这个顺序来背：乾兑离震巽坎艮坤对应 12345678。 3.后天八卦取数
这里依然有一段口诀：

戴九履一，左三右七

二四为肩，六八为足

转化成九宫格后如下图：
（图略）
这个九宫格很多朋友小学的时候应该都接触过，横竖斜三个数加起来都等于 15，也就是《烟波钓叟歌》里说的：“先需掌上排九宫，纵横十五在其中”

那么综合以上几点得出结论：

坎宫对应数字 1、6；

艮宫对应数字 5、7、8、10；

震宫对应数字 3、4、8；

巽宫对应数字 3、4、5、8；

离宫对应数字 2、3、7、9；

坤宫对应数字 2、5、8、10；

兑宫对应数字 2、4、7、9；

乾宫对应数字 1、4、6、9；

中宫对应数字 5、10（飞盘奇门中才会启用中宫）。总结如下图：

```

    3/4/5/8  |  2/3/7/9  |  2/5/8/10
    -------------------------
    3/4/8    |   5/10    |   2/4/7/9
    -------------------------
    5/7/8/10 |   1/6     |   1/4/6/9

```

那么在实践中如何取用呢？遵循以下几点原则：

1.一般多用后天八卦数，就是九宫数；其次是先天八卦数；应在五行数的情况较少，在定数字的时候，后天数与先天数结合起来，优先使用。

2.宫位旺相取大数，休囚取小数。

3.外盘主远主大取大数，内盘主近主小取小数。如起局为阴遁局，则离、坤、兑、乾为内盘，坎、艮、震、巽为外盘，若起局为阳遁局则反之。

```
阴遁局，则离、坤、兑、乾为内盘，坎、艮、震、巽为外盘
    外  |  内  |  内
    ------------------
    外  |      |  内
    ------------------
    外  |  外  |  内
```
