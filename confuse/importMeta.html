import.meta

它是esm语法。 由宿主环境提供以下属性

url resolve

api
url 返回此模块的完整url。（包括qs、碎片等）
resolve 将一个模块的标识符解析为当前模块的url

demo

<script type="module">
  import "./index.mjs?someURLInfo=5";
</script>

// index.mjs
new URL(import.meta.url).searchParams.get("someURLInfo"); // 5
