configuration types

配置文件的形式：

  • 输出一个返回对象的方法。该方法的参数是env、argv (env, argc) => object
  • 输出一个返回promise的方法。 () => Promise<object>
  • 多个配置文件。需要处理好：
    • 依赖关系。 dependencies:['moduleName']
    • 最大同时打包数量。 module.exports.parallelism = 1
