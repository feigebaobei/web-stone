# pwa
> Progressive Web Apps  
> 是一个网页。拥有app般的用户体验  
> 2015年提出。2016年推广。好像safri不支持  
> 

## feature
- 渐进增强  
- 无网可用  
- 类原生应用，可安装。
- 安全。使用https提供服务。
- sso友好
- 信息推送

## demo
1. 创建一个html页面。
2. 添加`manifest.json`文件  
3. 添加service worker（可实现无网运行）
